BlackLab is a Java search library on top of Apache Lucene, developed at The Dutch Language Institute. To get started with BlackLab, take a look at {@link nl.inl.blacklab.search.Searcher Searcher} (the main BlackLab API), {@link nl.inl.blacklab.search.TextPattern TextPattern} (to build abstract query objects, for example from a parser), {@link nl.inl.blacklab.search.Hits Hits} (results object; provides sorting functionality) and {@link nl.inl.blacklab.search.grouping.ResultsGrouper ResultsGrouper} (provides grouping functionality).

For more information, contact Jan Niestadt at the Dutch Language Institute (jan.niestadt@ivdnt.org)
